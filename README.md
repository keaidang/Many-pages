# 前端世界 (Frontend Showcase)

一个展示现代前端设计、工具和 HTML5 游戏的精选集。

## 项目结构

```
Many-pages/
├── css/                # 全局样式
│   └── style.css       # 主页样式
├── js/                 # 全局脚本
│   └── main.js         # 主页逻辑
├── designs/            # 独立设计案例
│   ├── earth/          # 3D 地球可视化
│   └── README.md       # 设计目录文档
├── index.html          # 主页入口
└── README.md           # 项目文档
```

## 功能特性

-   **精选设计**: 展示高质量的 UI/UX 案例（如 3D 地球）。
-   **前端工具**: 实用开发工具集合（占位符）。
-   **HTML5 游戏**: 纯 Web 技术实现的小游戏（占位符）。

## 快速开始

1.  克隆仓库。
2.  使用 Live Server 或任何静态文件服务器打开 `index.html`。
3.  尽情探索！

## 贡献

欢迎提交新的设计案例！请参考 `designs/README.md` 了解如何添加新设计。
